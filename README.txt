Tutoriel / concept : 

Votre but est d'obtenir le plus haut score en touchant les elements affichers dans le flipper au protogen, les objets raportent plus ou moins de points a votre compteur selon le classement ci contre :

> Oreo : 500pts

> Barre de RAM : 350pts

> Brique blanche : 100pts



Votre score s'affichera sur la page en haut a gauche, juste en dessous de notre nom de joueur 


Vous possèdez 3 vies, faite attention car une fois arriver a 0. La partie se terminera !
